# MuslimDesk — Free Islamic Prayer Times & Azan App for Windows

![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)
![Platform: Windows](https://img.shields.io/badge/platform-Windows-blue.svg)

**MuslimDesk** is a free Windows desktop application for Muslims. It brings accurate **prayer times (Salah/Namaz)**, automatic **Azan (Adhan) reminders**, the **Holy Qur'an**, **daily Azkar and Duas**, **Hijri calendar**, **Qibla direction**, a **Tasbih counter**, an alarm clock, a whiteboard, a PC-cleanup tool, and a **nearby mosque finder** to the Windows desktop.

The application runs quietly in the system tray, calculates prayer times fully offline, and can automatically play the Azan for the five daily prayers.

It is a companion project to the [IOM Daily Azkar](https://github.com/kmarsus/app-iom-daily-azkar) Android application.

## Download

This repository hosts downloads only. Grab the latest version from the Releases page:

**https://github.com/kmarsus/MuslimDesk/releases**

Open the latest release and download the Windows program from the **Assets** section — no installer needed.

## Main Features

- Accurate daily prayer times (Fajr, Sunrise, Dhuhr, Asr, Maghrib, Isha)
- Automatic Azan playback with multiple calculation methods
- Hanafi and standard Asr calculation options
- Country and city selection, or manual latitude/longitude
- Fully offline prayer-time calculation
- Custom adjustment for individual prayer times
- User-selectable Azan audio and a separate alarm-sound catalog
- Alarm clock with AM/PM time and custom sounds
- Floating always-on-top clock
- Whiteboard with drawing tools
- PC Clean tool for Temp/browser cache cleanup
- Qur'an, daily Azkar, Duas, Hijri calendar, Qibla direction, Tasbih counter, nearby mosque finder
- System tray operation, starts automatically with Windows
